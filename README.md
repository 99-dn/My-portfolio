# DNA — Portfolio Website
Personal portfolio for **DNA**, a Data & AI Automation Engineer based in Accra, Ghana 🇬🇭.  
Built to represent precision engineering — dark, terminal-inspired, zero templates.

## Live Site
🔗 https://my-portfolio-two-neon-22.vercel.app

## Tech Stack
- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (no frameworks)
- Google Fonts CDN (JetBrains Mono, Inter)

## Features
- Animated particle network hero
- Terminal typewriter effect
- Project filter by category
- Certificate modal viewer
- CV & Resume download
- Contact form via Formspree
- Fully responsive (mobile & desktop)

## Sections
- **Hero** — Name, role, tagline, CTA buttons
- **About** — Bio, skills, language goals
- **Projects** — Filterable project cards
- **Credentials** — Certificates, CV, Resume
- **Contact** — Formspree-powered contact form
